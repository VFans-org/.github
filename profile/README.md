## 欢迎大家成为VFans DAO的贡献者，踏上共同创建去中心化社区的征途 👋

### 使命与愿景
使命： 建立去中心化的创作者社区，建增加创作者收入，为用户持续产出高质量的内容。

愿景： 用DAO和区块链技术重塑创作者经济。

### VFans DAO的组成部分和价值创造 

DAO 是一个由代币所有权、政策和智能合约组成的虚拟组织。 VFans DAO 是一个营利性 DAO，旨在通过高质量的付费内容提高 DAO 成员的收入和社区价值。<br>
